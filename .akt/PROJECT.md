## Project Context

- **What it is:** YaLoSé ("I already know it") — an offline-first Spanish vocabulary flashcard PWA that uses spaced repetition and RAE definitions to help users master high-frequency words, hosted free on GitHub Pages.
- **Tech stack:** React 19, Vite 7, JavaScript (ESM), CSS modules, localStorage/IndexedDB, GitHub Actions, Puppeteer (build-time RAE scraper)
- **Key conventions:** functional components only, no classes; full vocabulary in `public/vocabulary.json` (5000 words, versioned by hash); runtime-fetched file is `public/vocabulary-slim.json` (~200 KB, generated by `prebuild` hook via `scripts/build_slim_vocabulary.js`); localStorage cache key `yalose-vocabulary-slim-cache`; user progress stored in localStorage; no backend — all integrations happen at build time via GitHub Actions
